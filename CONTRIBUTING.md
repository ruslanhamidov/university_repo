# Contributing to Our Project

We welcome all contributions! Please follow these steps to make the process smooth for everyone.

## 1. Fork the Repository
- Click the **Fork** button at the top-right of the repository page.
- Clone your fork locally:
  `git clone https://github.com/YOUR_USERNAME/REPO_NAME.git`

## 2. Create a Branch
Always create a new branch for your task:
- `git checkout -b type/short-description`
*(Types: feat, fix, docs, refactor, chore)*

## 3. Make Changes & Commit

- Write your code or make changes.
- Commit your changes using the convention:
  `git commit -m "type(scope): description"`

## 4. Push and Open a PR
- Push your branch: `git push -u origin your-branch-name`
- Go to the original repository on GitHub and click **Compare & pull request**.
- Add a clear description and link your Issue: `Closes #ISSUE_NUMBER`.

---
*If you have any questions, feel free to open an Issue or ask in the team chat!*
