# Git Glossary

## Branch
A separate line of development.
Example: feat/glossary

## Commit
A saved snapshot of changes.
Example: git commit -m "feat(glossary): add git terms"

## Fork
A personal copy of another repository.
Example: Forking a project to your GitHub account.

## Clone
Downloading a repository to your computer.
Example: git clone <repo-url>

## Pull Request
A request to merge changes into another branch.
Example: Opening a PR from feat/glossary to main.

## Merge
Combining changes from one branch into another.
Example: Merging a PR into main.

## Rebase
Moving your commits on top of another branch.
Example: git rebase origin/main

## Conflict
When Git cannot automatically merge changes.
Example: Two people edit the same line.

## Origin
Your forked repository remote.
Example: git push origin main

## Upstream
The original repository you forked from.
Example: git fetch upstream
